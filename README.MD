## Features

- Generate random harmonious color palettes
- Copy individual colors with one click
- Save and remove favorite palettes locally
- Export palette as:
    - CSS variables
    - Tailwind config
    - JSON
- Responsive layout for mobile, tablet and desktop
- Fast and lightweight

## Installation

Clone the repository:

```bash
git clone https://github.com/Jimi090/ColorGen.git
```

Go to the project directory:

```bash
cd colorGen
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Usage

1. Click Generate to create a new palette.

2. Click any color to copy its HEX value.

3. Click Add to favorites to save the current palette.

4. Open Favorites to view saved palettes.

5. Export your palette as CSS, Tailwind config or JSON.
